## 파일 구조

```
apps/landing/src/
├── components/
│   ├── Header.astro         # 헤더 (로고 + 내비 + Lab CTA + 모바일 메뉴)
│   └── Footer.astro         # 푸터 (4-컬럼 + 법적 고지)
├── layouts/
│   └── Base.astro           # 마스터 레이아웃 (meta + Typekit + Header + Footer)
├── lib/
│   ├── apps.ts              # StudioApp[] 데이터 (카드용)
│   └── app-details.ts       # AppDetail 데이터 (상세 페이지용)
├── pages/
│   ├── index.astro          # 홈
│   ├── about.astro          # 소개
│   ├── faq.astro            # FAQ
│   └── apps/
│       ├── index.astro      # 앱 목록
│       └── [app].astro      # 앱 상세 (동적)
├── styles/
│   └── tokens.css           # 디자인 토큰 (3-tier, oklch, 8pt grid)
└── env.d.ts                 # Astro 타입 선언
```
